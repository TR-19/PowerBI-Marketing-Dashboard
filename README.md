# Power BI Marketing Dashboard

## Overview
This project contains an interactive Power BI dashboard built using marketing and banking customer datasets.

## Datasets
- marketing_campaign.csv
- bank_customers_train.csv

## Features
- Total customer spending analysis
- Campaign response performance
- Product category insights
- Interactive visuals (cards, bar charts, pie charts, gauge)

## Tools Used
- Power BI Desktop
- CSV datasets

## How to Use
1. Download the .pbix file
2. Open it using Power BI Desktop
3. Refresh data if required
